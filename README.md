# WordUp Translation App

## Description

This repository is where we build the WordUp translation application to provide regional and pop-culture language translations for a given word.

## Installation

N/A

## Usage

Open the application in your browser and enter a word into the search bar to begin. You will be given a list of translations for that word, with options to learn more information. You can also access our Word of the Day and a history of your recent searches.
